# Wibecoded android web camera2windows

> all ffmpeg dlls in zip
> and manual copy softcam.dll to program path
> and file scrcpy-server copy to folder where you run this

# Sources
```
scrcpy-server from https://github.com/genymobile/scrcpy (from program folder)
ffmpeg sources in https://github.com/GyanD/codexffmpeg/releases/tag/8.1 ffmpeg-8.1-full_build-shared.zip file
```

# Where i run this
`tested on POCO X3 NFC (Pixel Experience) and Windows 10`

`webcamera app for discord XD`

# Program do
```
adb:
push scrcpy-server file to android temp directory
run port forwarding
and start scrcpy-server

Connecting to scrcpy-server (localhost:1234)
```
